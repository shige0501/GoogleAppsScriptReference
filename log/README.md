# ログの出力

## メソッド
Logger.log(data)          ログの出力
Logger.log(format, value) ログの出力

## 引数
data    ログ・メッセージ
format  ログ出力のフォーマット
values  フォーマットに渡す値

## 解説
フォーマット上で%sと指定することで、valuesの変数値をログに
埋め込むことができます。

## 戻り値
Logger
